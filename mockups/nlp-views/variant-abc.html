Was diese Variante kann (und was nicht)

✓ Das ist eine echte "wie sollte meine Sitemap aussehen"-Sicht: URL-Slugs, H1-Vorschläge, Status pro Page, Target-Queries.

✓ Du siehst sofort, wieviele Pages eigentlich fehlen (10 von 14 hier) — das war im aktuellen Entity-Graph nirgends sichtbar.

✓ "likely_exists_elsewhere" markiert Pages, die wahrscheinlich schon auf der Site sind aber wir aus der einen URL nicht sehen — Hinweis auf eine spätere Crawl-Integration.

! Größter Eingriff: Phase 6 im Prompt, ein neues Feld recommended_sitemap im Schema, neue UI-Komponente.

! Wichtige Annahme: das LLM darf hypothetisieren. URL-Slugs sind Vorschläge, kein Crawl-Ergebnis. User muss verstehen: das ist eine Empfehlung, kein Audit.
