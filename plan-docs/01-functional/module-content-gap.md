# Modul: Content Gap

Wird in Phase 5 erarbeitet. **Komplett neu**. Aus Main-Keywords werden SERP-Daten gezogen → Konkurrenten gesammelt → Ranking-Daten der Konkurrenten geholt → Gap dargestellt.

Kostenfaktor: SERP-API-Calls. Strategie für Caching/Re-Use ist Modul-kritisch.

Verknüpfungen:

- **Content Writing** (Gap → Brief)
- **Keyword Clustering** (Cluster + Gap → Themen-Priorisierung)
- **Strategy** (Gap-Findings können als `content`-Findings auflaufen)
