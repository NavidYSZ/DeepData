# Error Fix Log

Bugs, die im Lauf der Implementierung gefunden und behoben wurden. Pro Eintrag: Symptom, Root Cause, Fix, Prävention. Verhindert Wiederholfehler in zukünftigen Sessions.

Format:

```
## YYYY-MM-DD — <Modul/Bereich> — <Kurztitel>
- **Symptom:** was war beobachtbar?
- **Root Cause:** warum?
- **Fix:** was wurde geändert?
- **Prävention:** Regel/Check/Doc-Update, damit es nicht wiederkommt
- **Refs:** Commit, Plan-Doc, ADR
```

---

_Noch keine Einträge — Implementierung beginnt erst nach approved Modul-Specs._
