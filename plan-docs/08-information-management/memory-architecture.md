# Memory-Architektur (Notes-Modul)

Wird mit dem Notes-Modul-Spec erarbeitet. Da es in v2 keinen Chat gibt ([`../decisions/ADR-0008-no-chat-in-v2.md`](../decisions/ADR-0008-no-chat-in-v2.md)), ist Memory hier ein **reines User-Notizen-Modul** — keine Agent-Befüllung, keine Vector-Suche im MVP. Strukturierung in Kategorien (Fakt, Entscheidung, Notiz, Erinnerung), Verknüpfung zu Modul-Daten als Cross-Refs.
